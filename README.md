This is a simple todo application developed using Node JS.

## Functionalities
- Adding Tasks
- Deleting Tasks
- Reading Tasks
- Listing all the Tasks

## Libraries used
- loadash - manipulate arrays to save Tasks
- yargs - command line arguments
- nodemon - live server refresh
